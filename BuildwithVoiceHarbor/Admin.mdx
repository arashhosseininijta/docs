## Stage-based admin functionalities

<Steps>
  <Step title="1. Generate a developer token using your admin token">
    Use your **admin token** to create developer tokens (also called usage tokens) for your developer team.

    [API Reference → Developer Token Generation](https://docs.nijta.com/api-reference/admin/developer-token)

    ```bash
    curl -X POST https://voiceharbor.ai/api/admin/developer-token \
      -H "Authorization: Bearer YOUR_ADMIN_TOKEN"
    ```

    **Response:**
    ```json
    {
      "developerToken": "abc123-usage-token"
    }
    ```
  </Step>

  <Step title="2. View all developer tokens created under your admin token">
    Use your **admin token** to retrieve a list of all usage tokens you've generated.

    [API Reference → List Developer Tokens](https://docs.nijta.com/api-reference/admin/developer-tokens)

    ```bash
    curl -X GET https://voiceharbor.ai/api/admin/developer-tokens \
      -H "Authorization: Bearer YOUR_ADMIN_TOKEN"
    ```

    **Response:**
    ```json
    {
      "developerTokens": [
        {
          "usage_token": "abc123-usage-token",
          "created_at": "2025-08-28T12:34:56Z"
        }
      ]
    }
    ```
  </Step>

  <Step title="3. Monitor jobs and usage with developer tokens">
    Use your **developer token** to view jobs and job content.

    **List jobs:**
    ```bash
    curl -X GET https://voiceharbor.ai/api/jobs \
      -H "Authorization: Bearer YOUR_DEVELOPER_TOKEN"
    ```

    **Response:**
    ```json
    {
      "jobs": [
        {
          "job_id": "job-xyz",
          "token": "abc123-usage-token",
          "created_at": "2025-08-28T10:00:00Z"
        }
      ]
    }
    ```

    **Get job content:**
    ```bash
    curl -X GET https://voiceharbor.ai/api/jobs/JOB_ID/content \
      -H "Authorization: Bearer YOUR_DEVELOPER_TOKEN"
    ```

    **Response:**
    ```json
    {
      "jobContent": [
        {
          "id": 1,
          "job_id": "job-xyz",
          "file_name": "audio1.wav",
          "audio_duration": 123,
          "created_at": "2025-08-28T10:01:00Z"
        }
      ]
    }
    ```
  </Step>
</Steps>
